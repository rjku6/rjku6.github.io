intro

让我们在不到5分钟内了解Docusaurus。

开始​

通过创建一个新站点来开始。

或者立即使用docusaurus.new来尝试Docusaurus。

所需工具​

  • Node.js 版本需为16.14或更高:
    • 在安装Node.js时，建议您勾选与依赖相关的所有复选框。

生成新站点​

使用classic模板生成一个新的Docusaurus站点。

在运行以下命令后，classic模板将自动添加到您的项目中：

npm init docusaurus@latest my-website classic

您可以将此命令输入到命令提示符、PowerShell、终端或任何其他代码编辑器的集成终端中。

该命令还会安装您运行Docusaurus所需的所有必要依赖项。

启动站点​

运行开发服务器：

cd my-website
npm run start

cd命令用于更改您正在使用的目录。为了使用您新创建的Docusaurus站点工作，您需要导航到终端所在的目录。

npm run start命令会在本地构建您的网站，并通过开发服务器提供服务，您可以在 http://localhost:3000/ 上查看您的网站。

打开 docs/intro.md（本页）并编辑一些行：站点会自动重新加载并显示您的更改。
